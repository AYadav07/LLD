# LLD — Low Level Design repository

This repository will contain Low-Level Design (LLD) implementations and reference examples. Designs will be added incrementally. Each design should live under `designs/<feature>/` and include a `README.md`, diagrams (PlantUML or Mermaid), and optional `src/` and `tests/` directories.
